# Roman Keycard Blackwood (RKC 1430)

Used to count key cards (the four aces plus the king of trumps) before bidding slam.

---

## The Ask

**4NT** = asking for key cards (must have a clearly established trump suit)

The "1430" naming refers to the responses:

| Response | Key cards held |
|----------|---------------|
| 5♣ | 1 or 4 key cards |
| 5<span style="color:red;">♦</span> | 0 or 3 key cards |
| 5<span style="color:red;">♥</span> | 2 key cards, **no** trump queen |
| 5♠ | 2 key cards, **with** trump queen |
